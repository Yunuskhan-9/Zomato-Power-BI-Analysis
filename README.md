# Zomato-Power-BI-Analysis
This project presents an interactive Power BI dashboard built on Zomato’s restaurant dataset. It highlights key insights into restaurant distribution, cuisine popularity, customer ratings, and pricing strategies across different cities and countries.
🔑 Features
- Global restaurant distribution with map visuals
- Cuisine popularity and customer preference analysis
- Ratings vs. pricing trends for performance insights
- Top restaurants and cities highlighted with KPIs
- Interactive filters for deep dive exploration
🎯 Outcome
The dashboard transforms raw Zomato data into actionable business intelligence, helping identify market opportunities, customer trends, and competitive landscapes in the food-tech industry.
